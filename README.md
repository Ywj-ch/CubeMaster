# 魔方颜色与布局规范

## 世界魔方协会（WCA）标准配色
| 颜色 | 对面颜色 |
|------|----------|
| 白色（White） | 黄色（Yellow） |
| 红色（Red） | 橙色（Orange） |
| 蓝色（Blue） | 绿色（Green） |

### 面位缩写
- **U (Up)**：顶面（白色中心）  
- **R (Right)**：右面（蓝色中心）  
- **F (Front)**：前面（红色中心）  
- **D (Down)**：底面（黄色中心）  
- **L (Left)**：左面（绿色中心）  
- **B (Back)**：后面（橙色中心）

## 标准读取顺序
魔方状态数据按以下面位顺序读取/存储：  
`U（顶面）→ R（右面）→ F（前面）→ D（底面）→ L（左面）→ B（后面）`
